📥 IMPORT PLAYERS VIA CSV

📋 CSV FORMAT (first row = header):
name, role, age, base_price, nationality, image_filename
Columns:
• name — Player full name (required)
• role — Batsman / Bowler / All-Rounder / Wicket-Keeper
• age — Integer age
• base_price — Price in Crores (e.g. 1.5)
• nationality — Indian / Overseas etc.
• image_filename — filename of uploaded image (e.g. virat.jpg)
EXAMPLE ROW:
Virat Kohli, Batsman, 35, 20, Indian, virat.jpg
Jasprit Bumrah, Bowler, 30, 15, Indian, bumrah.png
